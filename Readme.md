#  AI-Based Skin Disease Detection System

An intelligent web application that uses **Deep Learning & Computer Vision** to detect skin diseases from images and generate instant reports.

---

##  Features

*  User Authentication (Login / Signup)
*  Real-time Image Upload / Webcam Detection
*  AI-based Disease Prediction
*  Detailed Report Generation
*  Email Report Sending System
*  Fast & User-Friendly Interface

---

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **Machine Learning:** TensorFlow / Keras
* **Database:** SQLite
* **Libraries:** OpenCV, NumPy, Pandas

---

##  How It Works

1. User uploads an image or uses webcam
2. Image is preprocessed using OpenCV
3. Deep Learning model analyzes the image
4. System predicts the skin disease
5. Report is generated and displayed/sent via email

##  Installation & Setup

### 1️ Clone the repository

```
git clone https://github.com/your-username/AI-Based-Skin-Disease-Detection.git
cd AI-Based-Skin-Disease-Detection
```

### 2️ Create virtual environment

```
python -m venv venv
venv\Scripts\activate
```

### 3️ Install dependencies

```
pip install -r requirements.txt
```

### 4️ Run the application

```
python app.py
```

 Open in browser:

```
http://127.0.0.1:5000/
```
